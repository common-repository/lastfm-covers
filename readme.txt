With this plugin you can show your recent or favorite LastFM Tracks on your sidebar

== Description ==

This plugin allow bloggers to display a nice "LastFM widget" that shows the last tracks scrobbled by LastFM.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `lastfm` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the plugin configuration in the wordpress "settings" category
4. Add the widget in your sidebar
